Heartist - Heart Assistant Application

Data Analysis Project | Feb 2025 - Mar 2025

Project Details

Created a Heart assistant application with Python and various libraries to fetch data from an ECG and convert it into actionable insights.

3rd Place in Inter-College Crossroads Competition (50+ teams)

Key Features

  • Used Matplotlib in Python to analyze and compare 100+ ECG readings, assessing heart functionality with visual insights
  • Converted raw PDFs into actionable insights, accurately detecting and highlighting Q, R, S waves in ECG reports with 76%+ precision
  • Loaded and preprocessed 500+ ECGs using Python & NumPy, enhancing signal clarity and reducing noise by 40%
  • Developed intuitive visualizations to help medical professionals quickly interpret complex ECG data

Technologies Used

Python NumPy Pandas Matplotlib Data Analysis ECG Processing
